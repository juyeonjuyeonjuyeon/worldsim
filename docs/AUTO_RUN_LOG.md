#### 녹색 섬광 (Green Flash)
- `_prev_sun_elev` / `_green_flash_t` 변수 추가

#### 과잉호 (Supernumerary Rainbow)
- 무지개 셰이더에 `supernumerary_str` uniform 추가
- 40.6°-36.5° 구간 파동 간섭 패턴: cos((ang-36.5)/1.5 × 6.28)
- 작은 물방울(안개비)일수록 강하게, 굵은 물방울일수록 약하게

#### 달무지개 (Moonbow)
- 동일 Descartes 각도(40.6°~42.5°), moon_dir 기준
- 최대 강도 0.04 (태양무지개의 4%)
- 반달 이상 + 태양 -5° 이하 + 비 이력 조건

#### 안개무지개 (Fogbow)
- 안개 밀도 > 0.003 + 비 없음 + 태양 가시 조건
- 34°~43° 넓은 흰색 호 (각도가 더 넓고 색이 없음)
- 별도 fogbow 셰이더 (색 없이 흰색)

#### 황도광 (Zodiacal Light)
- 태양 방위각 ±35° 원뿔 방향
- 태양 -1° ~ -25° 구간 박명 조건
- 구름 Beer-Lambert 억제

#### 대기광 (Airglow)
- 태양 -18° 이하 깊은 밤 조건
- OI 557.7nm (녹색), OH 밴드 — night_horizon/night_top에 직접 합산
- `Color(0.003, 0.012, 0.005)` × airglow_t

#### 은하수 (Milky Way)
- PrimitiveSphere r=399, mw_shader (is_shader_type spatial, blend_add)
- 은하 북극(J2000 RA=192.859°, Dec=27.129°) + 은하 중심(RA=266.405°, Dec=-28.936°)
- 세차 보정 + AltAz 변환 → gal_pole/gal_center 셰이더 파라미터
- 태양 -10° 이하 + clear_sky 조건으로 intensity 0.20 fade in

#### 행성 합/충 표시

  ① 태양 글레어 헤일로가 달 원반 바깥(d>0.11)으로 뻗어 가려지지 않아 밝은 흰 공으로
  잔존 ② 대기 셰이더 태양방향 Mie 전방산란 피크가 u_sun_intensity 잔여로 밝게 남음.
- 수정: ① 태양 셰이더 glare_atten=1-cover로 글레어 전체를 가시 광구 비율로 감쇠
  (개기 시 0) ② u_sun_intensity=1-cover (개기 시 정확히 0 → Mie 피크 소멸).
  정상 하늘(cover=0)은 1.0 그대로 — 회귀 없음.
- 검수: 개기=검은 원반+진주빛 코로나+어두운 하늘(상징적 모습), 부분=하늘 감광+크레센트.

### A4. 검수 하네스 개선 (커밋 82cec2f에 포함, 재사용 가능)
- `--look-sun`/`--look-moon`: 천체 방위·고도로 카메라 자동 조준(alt/az→월드 방향→yaw/pitch).
- 카메라 조준 후 `_update_all` 재호출: 빌보드 카브 투영(eclipse_moon_uv)을 최종 시점으로
  재계산. (없으면 수렴 루프 때의 기본 카메라 기준값이 남아 디스크에서 빗나감.)

## C. 자동 날씨 생성 — 현재 구현 검수 (사용자 요청)

### 현재 구조 (climatology sampler)
1. **기후표**(Weather.gd): 7 위도대×12월의 강수확률·눈경향·강수강도·구름량. 위도 양선형 보간.
2. **경도/지역 보정**(_get_climate_bias): 몬순(남아시아) 등 — precip_mult·amp_mult·mean_offset.
3. **확률적 전이**(_roll_auto_weather): 1~4 시뮬일마다 새 상태 추첨. 강수확률→비/눈(온도로
   눈경향 보정), 아니면 구름량→흐림/적운/권운/맑음 계층.
4. **기온**(_estimate_temperature): 위도 연평균+계절 코사인(8월 피크)+일교차(14시 피크)+
   날씨보정+남반구 반전+경도 바이어스.
5. **습도**(Environment.gd): 위도 기준+일변화+지면수분+날씨별 타깃.
6. **바람**: 위도별 탁월풍+날씨별 편차폭·풍속범위. move_toward로 부드러운 전이.

### 강점
- 실측값 기반(주석에 측정치 인용), 좌표·계절·경도(지역) 차등 = "좌표별·계절별" 충족.
- 온도-강수 결합(눈/비 판정), 1~4일 지속(자기상관) = 현실적 분포·지속성.

### 한계 — "실제 발생 원리"와의 간극
현재는 **위치·계절별 통계 분포를 올바르게 샘플링**하지만, 날씨가 **인과적으로 창발하는
역학계는 아님**. 실제 날씨 발생 원리에서 빠진 것:
1. **종관계(고·저기압·전선)의 이류 없음** — 공간 기압장·전선 통과 시퀀스(권운→고층운→
   난층운→강수→갠 하늘, 한랭전선)가 없음. 각 추첨이 독립(지속타이머 외 무기억).
2. **기압-지형풍 결합 없음** — 풍향이 기압경도가 아닌 탁월풍+잡음. 전선 통과 시 풍향
   backing/veering 없음.
3. **기단 연속성 없음** — 전선 통과 시 온도·습도 점프(한기/난기 부문)가 아닌 매끈한 일변화.
4. **지형·해양 근접 효과 미흡** — 해안선/고도 데이터 없음(해풍·비그늘·호수효과 부재).
5. **주간 대류 미반영** — 여름 오후 적운 발달→뇌우(CAPE)가 시각 구동 아님.

### 권장 업그레이드 (단순하지만 인과 보존 — CLAUDE.md 철학)
**종관 마르코프 + 이동 기압장**:
- 각 위치에 느리게 변하는 합성 기압(중위도 제트류로 W→E 이동하는 저·고기압 중심들의 합).
- 기압 경향에서 구름·강수 유도(하강→구름 두꺼워짐→전선강수, 상승→갬), 풍향은 기압
  경도(지형풍)에서 유도 → 풍향 veering 자연 발생.
- 날씨를 독립 추첨이 아닌 **전선 접근/통과 시퀀스**로. 전선 통과 시 온도·습도 점프.
- 기존 기후표는 합성 기압계의 빈도·강도를 위도·계절·지역별로 편향하는 **통계 외피**로 유지.
- 가벼움(이동 기압중심 몇 개+이류)으로 모바일 기준선 유지 가능. → **구름 작업과 함께 진행 권장.**

## D. 수증기흡수·신기루·편광 — 분석 (구현은 렌더 복구 후)

- **수증기 흡수**: H2O는 가시 적·주황(~590·650·720nm)에 약한 흡수, 근적외에 강한 흡수.
  가시영역 효과는 미묘(습한 대기에서 일몰 적색 약간 심화). **현 모델은 습도가 이미 Mie
  혼탁도(흡습성 에어로졸 성장)를 구동해 지배적 효과를 포착함.** 순수 H2O 흡수는 2차적.
  구현안: 오존처럼 적색 편중 흡수계수를 습도(가강수량)로 스케일해 소광(tau)에 추가. 저우선.
- **신기루**: 지표 수직 온도경사에 의한 굴절. 더운 지면→하방 신기루(도로 물웅덩이),
  역전층→상방 신기루(Fata Morgana). 지표/지평선 효과(하늘 돔 아님). 눈높이 하늘 sim에선
  더운 맑은 날 지평선 띠의 아지랑이/거짓 물면 정도. |지면온도−기온|로 게이팅한 화면공간
  수평 왜곡으로 구현 가능하나 **하늘 sim 가치 낮음 → 보류/최소 구현 권장.**
- **편광**: Rayleigh 산란은 태양 90°에서 최대 ~75% 편광. 그러나 **사람 눈은 편광을 거의
  볼 수 없음**(Haidinger 브러시 정도) → 맨눈 sim에 직접 렌더할 시각 효과 없음. 편광 필터/
  편광선글라스로만 90° 띠 어두워짐이 보임. **권장: 기본 렌더에 추가하지 않음(맨눈 불가시).
  선택적 "편광 필터" 데모 토글만 교육용으로.** = 의도적 미구현.

## E. 구름 작업 1단계 (커밋 5cad55a) — 사용자 승인 "코드 먼저, 검수 나중"

기존: 2D FBM 평면 + 날씨별 프리셋(권운 이방성 줄기/적운 뭉게/난층운·비 어두운 덩어리).
이미 **날씨별 형태·색**은 구현돼 있었음. 이번에 빠져 있던 부분 추가:
1. **좌표·계절별 형태 변조**: warm(적도1→극0)·summer(국지, 남반구 반전)로 대류 활성도
   계산 → 적운 셀 크기(noise_scale) 최대 35% 확대, 운저 고도 변조. 비대류형(권운·층운·
   비·눈)은 약하게(0.18) 변조해 운형 특성 보존.
2. **노을·여명 구름 적화**(셰이더): 태양 저고도(0~17°)에 구름 밑면이 적화된 직사광을
   받아 분홍·주황 underlit. 높은 태양 무영향.
- 안전성: 야간 ALPHA 가드(brightness 게이트) 미변경 → 밤 암막 회귀 불가. 주간 무영향.
  헤드리스 파스·런타임 통과. **⚠ 시각 자가검수는 윈도우 렌더 차단으로 보류 — 강도값
  사람 확인 후 튜닝 필요.**

### 구름 후속 (렌더 복구 후 우선순위)
- 노을 적화·셀크기 강도값 시각 튜닝.
- 구름 조명을 하늘 산란색에서 샘플(현재 sun_color×고정 ambient) → 박명 청회색·천정 반사광.
